## 📄 How to Use It

### 1. Summarize text
Paste text into the app or open a file. Then choose the summary option. The app will turn long content into a shorter version you can read fast.

### 2. Work with PDFs
Open a PDF file and let the app read the text inside it. It can handle reports, articles, and other common document types.

### 3. Open Word documents
Use `.docx` files to summarize notes, letters, and written documents without copying the text by hand.

### 4. Translate content
Choose a target language and translate the text or summary. This helps when you need to read content in another language.

### 5. Get key points
Use the key points view when you want the main ideas only. This is useful for study notes and quick reviews.

### 6. Play audio
Use text-to-speech to hear the summary or selected text. This is helpful when you want to listen instead of read.
